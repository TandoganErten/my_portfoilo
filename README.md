# Tandogan Erten - Portfolio Website

Professional portfolio website with multi-language support (English & Turkish).

## Features

- ✅ Multi-page navigation (Home, About, Skills, Projects, Contact)
- ✅ Bilingual support (English/Turkish)
- ✅ Responsive design
- ✅ SEO optimized
- ✅ WhatsApp integration
- ✅ Video portfolio
- ✅ Professional animations

## Language System

The website supports two languages:
- **English (EN)** - Default
- **Turkish (TR)**

### How It Works:
1. Click EN/TR buttons in the navigation bar
2. Language preference is saved in browser
3. All text updates automatically
4. SEO meta tags update for each language

### Adding/Editing Translations:
Edit the `translations.js` file:
```javascript
translations.en.home.title = "Your English Text"
translations.tr.home.title = "Türkçe Metniniz"
```

## Customization

### Update Contact Information:
In `contact.html`, update:
- WhatsApp number: `https://wa.me/905XXXXXXXXX`
- Email: `contact@tandoganerten.com`

### Add Your Social Media Links:
Update links in all HTML files:
- GitHub: `https://github.com/TandoganErten`
- LinkedIn: Your LinkedIn URL
- Twitter: Your Twitter URL

### Update Profile Image:
Replace `images/profile.jpg` with your photo

### Add Projects:
Edit `projects.html` to add your actual projects
